# HelloPrompt
Prompt some informations on your screen when opened terminal.

# Prerequirements
+ git 2.31.1
+ gradle 7.0.2
+ opanjdk 15.0.2

# Build
```
$ git clone https://github.com/asas1asas200/HelloPrompt.git
$ cd HelloPrompt
$ gradle build
$ gradle run # or run in quiet mode $ gradle run -q 
```
